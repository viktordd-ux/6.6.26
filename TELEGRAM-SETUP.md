# Настройка Telegram-бота для формы RSVP

Данные из формы уже отправляются в Telegram — нужно только создать бота и указать его данные в форме.

## Шаг 1: Создать бота и получить токен

1. Откройте Telegram и найдите **@BotFather**.
2. Отправьте команду: `/newbot`.
3. Введите имя бота (например: «Свадьба Даниил и Софья») и username (должен заканчиваться на `bot`, например: `svadba_rsvp_bot`).
4. BotFather пришлёт **токен** вида: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`.  
   Скопируйте его — это ваш **Telegram Bot Token**.

## Шаг 2: Узнать Chat ID (куда приходят сообщения)

Сообщения можно получать в **личный чат** с ботом или в **группу/канал**.

### Вариант А: Личный чат с ботом

1. Напишите вашему боту любое сообщение (например: «Привет»).
2. Откройте в браузере (подставьте свой токен):
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
3. В ответе найдите `"chat":{"id": 123456789}` — число `123456789` и есть **Chat ID**.  
   Обычно для личного чата это положительное число.

### Вариант Б: Группа

1. Создайте группу в Telegram и добавьте туда вашего бота (как участника).
2. Напишите в группе любое сообщение.
3. Снова откройте `getUpdates` (см. выше) — в ответе будет `"chat":{"id": -1234567890}`.  
   У групп **Chat ID отрицательный** — скопируйте его целиком (вместе с минусом).

### Вариант В: Темы в группе (опционально)

Если в группе включены **темы**, у каждой темы есть свой **message_thread_id**.  
В интерфейсе группы: правая кнопка по теме → «Копировать ссылку» — в ссылке будет число, это и есть ID темы.  
Либо посмотрите в `getUpdates` после сообщения в нужной теме: поле `message_thread_id`.

## Шаг 3: Вписать данные в форму

Откройте файл **`index.html`** и найдите форму RSVP (примерно строки 349–351). Сейчас там:

```html
data-telegram-bot-token=""
data-telegram-chat-id=""
data-telegram-thread-id=""
```

Замените на свои значения:

- **data-telegram-bot-token** — токен от BotFather (обязательно).
- **data-telegram-chat-id** — Chat ID из шага 2 (обязательно).
- **data-telegram-thread-id** — ID темы в группе или оставьте пустым, если темы не используете.

Пример:

```html
data-telegram-bot-token="1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"
data-telegram-chat-id="-1001234567890"
data-telegram-thread-id=""
```

Сохраните файл. После этого при отправке формы вы будете получать в Telegram сообщение вида:

```
Новая RSVP-анкета

Имя: ...
Статус присутствия: Да / Нет
Имя спутника: ...
Количество гостей: ...
Предпочтения по еде: ...
Предпочтения по алкоголю: ...
Вопросы/пожелания: ...
```

### Если в форме появятся новые поля

Чтобы новые поля формы попадали в Telegram:

1. **index.html** — добавьте поле в форму (input/textarea/select с атрибутом `name="rsvp_название"`).
2. **js/script.js** — в трёх местах:
   - объявите переменную для поля (как `nameInput`, `guestsInput` и т.д.);
   - в объекте **formData** при отправке добавьте значение нового поля;
   - в **buildTelegramMessage** добавьте строку в массив (например: `"Название поля: " + (payload.название || "—")`);
   - при необходимости добавьте проверку в **validateForm** и имя поля в **clearAllErrors**; если у поля есть `data-error-for`, добавьте его в список полей с `addEventListener("input", ...)`.
3. После правок пример сообщения в этой инструкции будет соответствовать тому, что приходит в Telegram.

## Важно

- Токен бота в HTML виден всем, кто откроет исходный код страницы. Для публичного сайта в перспективе лучше отправлять форму через свой сервер (backend), а токен хранить только на сервере. Для свадебного сайта часто оставляют как есть.
- Не публикуйте токен в открытых репозиториях. Если случайно залили — отзовите токен в BotFather (`/revoke`) и создайте нового бота.

После настройки просто откройте сайт, заполните форму и отправьте — сообщение должно прийти в указанный чат/группу.
